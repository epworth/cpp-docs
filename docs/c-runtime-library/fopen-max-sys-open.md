# `FOPEN_MAX`, `_SYS_OPEN`

## Syntax

```C
#include <stdio.h>
```

## Remarks

 `FOPEN_MAX` and `_SYS_OPEN` are the maximum number of files that can be opened simultaneously. `FOPEN_MAX` is the ANSI-compatible name. `_SYS_OPEN` is provided for compatibility with existing code.
